## 4. PHASE 2: HYBRID BOOKING

### 4.1 Purpose
Add full booking capability. Users can search, select seats, pay, and receive digital tickets – all in one flow.

### 4.2 Booking Flow (4 Clicks)

| Click | Action | Time |
|-------|--------|------|
| **1** | Search from → to → date. Results appear instantly. | 5 seconds |
| **2** | Select a bus from results list. | 3 seconds |
| **3** | Pick a seat from interactive seat map. | 5 seconds |
| **4** | Confirm booking (auto-filled passenger details) → Pay → Done! | 10 seconds |

**Total Time:** Under 30 seconds from search to confirmed booking.

### 4.3 Detailed Booking Steps

#### Step 1: Search
```
User enters:
- Departure: Lagos
- Destination: Abuja
- Date: 2026-06-25

Results:
┌─────────────────────────────────────────────┐
│ 🟢 GIG Motors    ₦8,000   6:00 AM   ★4.5   │
│ 🟡 ABC Transport ₦9,000   8:30 AM   ★4.2   │
│ 🔴 God is Good   ₦7,500   10:00 AM  ★3.8   │
└─────────────────────────────────────────────┘

Filters: Price range, time, rating
```

#### Step 2: Select Bus
```
User clicks "Book Now" on GIG Motors

Details shown:
- Company: GIG Motors
- Route: Lagos → Abuja
- Date: 2026-06-25
- Departure: 6:00 AM
- Price: ₦8,000
- Bus Type: Luxury Air-Conditioned
- Available Seats: 45/60

User clicks "Select Seat"
```

#### Step 3: Seat Selection
```
Interactive seat map appears:

┌────────────────────────────┐
│  [A1] [A2] [A3] [A4]       │ ← Available (green)
│  [B1] [B2] [B3] [B4]       │
│  [C1] [C2] [C3] [C4]       │
│  [D1] [D2] [D3] [D4]       │
│  [E1] [E2] [E3] [E4]       │ ← Taken (red)
│  [F1] [F2] [F3] [F4]       │
│                             │
│  🟢 Available   🔴 Taken    │
└────────────────────────────┘

User clicks available seat → Seat reserved for 5 minutes
User clicks "Book This Seat"
```

#### Step 4: Confirm & Pay
```
Passenger details auto-filled (saved from previous):
- Name: John Doe
- Phone: 08012345678
- Email: john@email.com

Booking Summary:
- Company: GIG Motors
- Route: Lagos → Abuja
- Date: 2026-06-25
- Time: 6:00 AM
- Seat: A3
- Price: ₦8,000
- Total: ₦8,000 (no unlock fee)

User clicks "Confirm & Pay" → Paystack popup → Payment
```

#### Step 5: Digital Ticket
```
After successful payment:

┌─────────────────────────────────────────────┐
│         ARRIVELINK TICKET                    │
│                                              │
│  Company: GIG Motors                        │
│  Route: Lagos → Abuja                       │
│  Date: 2026-06-25                          │
│  Time: 6:00 AM                             │
│  Seat: A3                                  │
│  Passenger: John Doe                       │
│                                              │
│  ┌──────────────────────┐                   │
│  │   [QR CODE]          │                   │
│  └──────────────────────┘                   │
│                                              │
│  Ticket Code: ARV-2026-06-25-0042          │
│  Status: ✅ Confirmed                       │
└─────────────────────────────────────────────┘

Ticket saved in user dashboard
Email + WhatsApp confirmation sent
Booking appears in company dashboard
```

#### Step 6: Company Confirms
```
Company sees booking in business dashboard:
- Booking ID: ARV-2026-06-25-0042
- Passenger: John Doe
- Seat: A3
- Status: Pending → Company clicks "Confirm"

User receives notification:
"Your booking with GIG Motors has been confirmed!"

If company cancels:
User receives notification and full refund
```
